# 컴퓨터 5대 구성요소

## 입력 장치 (Input Device)
- 키보드
- 마우스
- 마이크
- 스캐너

## 출력 장치 (Output Device)
- 모니터
- 프린터
- 스피커

## 중앙처리장치 (CPU)
- 산술논리연산장치(ALU)
    - 산술연산
    - 논리연산
- 제어장치(CU)
    - 명령어 해석
    - 실행 제어

## 기억 장치 (Memory)
- 주기억장치
    - RAM (임시 저장)
    - ROM (영구 저장)
- 보조기억장치
    - HDD
    - SSD

## 시스템 버스 (System Bus)
- 데이터 버스
    - 데이터 전송
- 주소 버스
    - 주소 정보 전달
- 제어 버스
    - 제어 신호 전달
